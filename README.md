💰 Expense Tracker

📖 Descrição

Projeto desenvolvido na Sprint 7 da TripleTen, com foco na implementação da lógica em JavaScript para um Rastreador de Despesas.

O HTML e CSS foram fornecidos, e o objetivo foi estruturar toda a lógica da aplicação e conectar os cálculos à interface.

🛠️ Tecnologias
• JavaScript (ES6)
• Manipulação de DO
• Eventos
• Local Storage

🚀 Funcionalidades
• Definição de orçamento
• Cálculo automático de:
• Total de despesas
• Média
• Saldo

• Alteração dinâmica da cor do saldo
• Estatísticas por categoria
• Identificação da maior categoria
• Adição e remoção de despesas
